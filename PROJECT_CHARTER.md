# Project Charter

## Governance

- No composite scoring. This project displays feed ratings verbatim only.
- No undisclosed commercial relationships with listed protocols or feed providers.
- All conflicts must be declared in this charter.
- Community members can submit data corrections via pull requests.

## Steward

Named steward to be identified and confirmed before final milestone delivery.

## Scope

**In scope v1:**
- Ethereum mainnet DeFi protocols (top 20 by TVL)
- Data ingestion from all major risk feeds
- Live TVL display
- Community-correctable data layer
- REST API for programmatic access

**Out of scope v1:**
- Cross-chain protocols
- Composite risk scoring
- Financial advice or recommendations
